# Future Improvements

This document lists possible enhancements that can be implemented in future versions of this project.

---

# Flight Controller

- Improve PID tuning for heavier payloads.
- Add runtime PID tuning through the transmitter.
- Implement flight mode switching.
- Support multiple PID profiles.

---

# Sensors

- Upgrade from MPU6050 to ICM-42688 or MPU9250.
- Add magnetometer support for heading hold.
- Add barometer for altitude hold.
- Add GPS for position hold and Return-to-Home.

---

# Software

- Improve complementary filter tuning.
- Add failsafe handling.
- Improve battery monitoring.
- Add telemetry support.

---

# Flight Performance

- Test larger batteries.
- Test different motor KV ratings.
- Compare 9×4.5, 10×4.5 and 11×4.5 propellers.
- Evaluate larger ESCs.
- Investigate payload capability.

---

# Hardware

- Design a custom PCB.
- Improve vibration isolation.
- Add power distribution board.
- Improve cable management.

---

s
Develop a fully custom Arduino-based flight controller with advanced flight modes while maintaining the educational simplicity of the YMFC-AL project.
